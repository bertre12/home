Ваша задача состоит:

▪️ Создать сигналы при удалении или создании записей в админ-панели.

▪️ Создать сигнал, который срабатывает при создании суперпользователя.

-----------Решение-----------

1. Создаём проект с названием - home

2. Создаём приложение  с названием - residents

3. Регистрируем приложение в файле - home/settings.py

4. Создаём модели в файле - residents/models.py

5. Регистрируем модель в файле - residents/admin.py

6. Создаём файл signals.py в папке residents
		(residents/signals.py)

7. В файле signals.py прописываем логику сигналов.

8. Регистрируем сигналы в файле residents/apps.py

9. Создаём миграцию.

10. Создаём суперпользователся с атрибутами:
			Имя пользователя - admin
			Пароль - admin
